Text Styles

Default font-family is Times New Roman.

1em represents the current font-size. If not already set, the default is the font-size of the document (html element) 16px.

H1: Main page heading

  • Font-weight: bold (700)
  • Font-size: 2em (32px)
  • Margin-top-bottom: 0.67em (21.4px)

Float

(interactive)

Float: left
Float: right

When an element is given the float property, it is lifted up out of the document flow. Following elements will then shift upward to take it's place. Then the floated element is placed back down, without moving the other elements. This allows following elements to intersect with, or be in the same place as, the floating element (ie this paragraph border is intersecting the boxes). However, text has a special property, it will wrap around the floating element. Fellow floating elements will not intersect each other, but will act similar to inline-block elements, either aligned left or right depending on the float. If an element is given the clear property, it will refuse to intersect floating elements, creating a new line.
